RouteHub

Bus Transportation Platform

React FastAPI PostgreSQL Docker JWT Spec-Driven Full-Stack
View on GitHub →

Digital platform for managing, scheduling, and monitoring institutional bus transportation. Supports user registration & auth, trip scheduling & booking, operational check-in, PDF reports, and in-app/email notifications — serving students, staff, drivers, and administrators.

  • Frontend: React 18 + Vite, MSW for browser-based development without backend
  • Backend: FastAPI, SQLAlchemy ORM, Alembic migrations, JWT auth, rate limiting
  • Architecture: Spec-driven — specs/ defines every user story, API endpoint, and edge case
  • Stack: HTML, JavaScript, CSS, Python, Gherkin, Docker
