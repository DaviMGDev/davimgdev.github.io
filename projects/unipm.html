unipm

Universal Package Manager

Go CLI Cross-Platform Distrobox
View on GitHub →

A meta package manager unifying apt, npm, pip, flatpak, brew, pacman, and Distrobox containers under a single CLI. Search, install, and remove software across ecosystems without memorizing per-tool flags.

  • Interactive source selection when packages exist in multiple ecosystems
  • Gherkin-based BDD test suite
  • Shell scripting for AppImage and containerized workflows
