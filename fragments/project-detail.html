my-agent

Go · LLM · Tool-Calling · Streaming · Mock Implementation

A Go-based LLM agent framework defining a minimal provider-agnostic interface with chat completion, tool-calling agent loop, and built-in mock implementation for testing without API keys.

Features a generic LLM interface (Chat, Complete, StreamChat), a tool-calling agent that orchestrates multi-step interactions, and MockLLM that echoes user input — great for prototyping and TDD. Streaming support via iterator pattern (ChatStream).

unipm

Go · CLI · Cross-Platform · Distrobox

A meta package manager unifying apt, npm, pip, flatpak, brew, pacman, and Distrobox containers under a single CLI. Search, install, and remove software across ecosystems without memorizing per-tool flags.

Features interactive source selection when packages exist in multiple ecosystems, Gherkin-based BDD test suite, and shell scripting for AppImage and containerized workflows.
