my-agent

LLM Agent Framework

Go LLM Tool-Calling Streaming Mock
View on GitHub →

A Go-based LLM agent framework defining a minimal provider-agnostic interface with chat completion, tool-calling agent loop, and built-in mock implementation for testing without API keys.

  • Generic LLM interface: Chat, Complete, StreamChat
  • Tool-calling agent that orchestrates multi-step interactions
  • MockLLM echoes user input — great for prototyping and TDD
  • Streaming support via iterator pattern (ChatStream)
  • CI pipeline with GitHub Actions
