### [RouteHub](https://github.com/DaviMGDev/RouteHub) — Bus Transportation Platform
> *Full-Stack · React · FastAPI · PostgreSQL · Docker · JWT · Spec-Driven*

Digital platform for managing, scheduling, and monitoring institutional bus transportation.
Supports user registration & auth, trip scheduling & booking, operational check-in, PDF reports,
and in-app/email notifications — serving students, staff, drivers, and administrators.

- **Architecture:** Spec-driven (`specs/` defines every user story, API endpoint, and edge case)
- **Stack:** HTML, JavaScript, CSS, Python, Gherkin, Docker

### [my-agent](https://github.com/DaviMGDev/my-agent) — LLM Agent Framework
> *Go · LLM · Tool-Calling · Streaming · Mock Implementation*

A Go-based LLM agent framework defining a minimal provider-agnostic interface with chat
completion, tool-calling agent loop, and built-in mock implementation for testing without
API keys.

- Generic `LLM` interface: `Chat`, `Complete`, `StreamChat`
- Tool-calling agent that orchestrates multi-step interactions
- `MockLLM` echoes user input — great for prototyping and TDD
- Streaming support via iterator pattern (`ChatStream`)
- CI pipeline with GitHub Actions

---

### [unipm](https://github.com/DaviMGDev/unipm) — Universal Package Manager
> *Go · CLI · Cross-Platform · Distrobox*

A meta package manager unifying `apt`, `npm`, `pip`, `flatpak`, `brew`, `pacman`, and
Distrobox containers under a single CLI. Search, install, and remove software across ecosystems
without memorizing per-tool flags.

- Interactive source selection when packages exist in multiple ecosystems
- Gherkin-based BDD test suite
- Shell scripting for AppImage and containerized workflows
